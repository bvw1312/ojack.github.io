PIXELSYNTH

30. March 2016

##Live Version

##Code

Browser-based synthesizer for creating sounds from images and drawings, programmed using javascript and Web Audio. PIXELSYNTH is inspired by the analog ANS Synthesizer created by Evgeny Murzin in 1937 for composing musical scores.
